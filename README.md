# Agenda de Servicios

## Descripción
Proyecto de práctica en Python para registrar servicios, calcular ingresos y visualizar ganancias por categoría.  
Permite agregar servicios con fecha, nombre y precio, mostrar todos los registros, calcular el total ganado y ver ganancias acumuladas por servicio.

## Tecnologías
- Python
- Listas y diccionarios
- Archivos de texto (.txt)
- Menú interactivo

## Cómo usar
1. Clonar el repositorio o descargar el archivo `agenda_servicios.py`.
2. Ejecutar el programa:
3. Seguir menu para agregar servicios y ver registros
